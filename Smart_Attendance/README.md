# Smart Attendance System using QR Code

This project is a Smart Attendance System that automates attendance marking
using QR Code technology. The teacher controls the attendance process by
generating a QR code, and students scan it to mark their attendance.

---

## 📌 Project Objective
To eliminate manual attendance, reduce proxy attendance, and provide
a secure and efficient attendance system using QR codes.

---

## 👩‍🏫 Teacher Module
- Generates a unique QR code for each class session
- QR code is valid only for a limited time
- Prevents fake or reused QR codes
- Controls the attendance process

---

## 👨‍🎓 Student Module
- Student enters their name
- Scans the teacher-generated QR code
- Attendance is marked automatically
- Duplicate attendance is prevented

---

## 🔐 Security Features
- Time-limited QR code
- Session-based validation
- Duplicate scan prevention

---

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- QRCode.js
- HTML5 QR Scanner
